# GitHub Pages Setup Instructions

Follow these steps to get your MFC Orders app online:

## Step 1: Upload Files to GitHub

You should have already created a repository. Now:

1. On your repository page, click **"Add file"** â†’ **"Upload files"**
2. Drag ALL these files into the upload area:
   - index.html
   - styles.css
   - app.js
   - pwa.js
   - sw.js
   - manifest.json
   - icon-192.svg
   - icon-512.svg
   - README.md

3. Scroll down and click **"Commit changes"**

## Step 2: Enable GitHub Pages

1. Click **"Settings"** (top of your repository page)
2. Click **"Pages"** in the left sidebar
3. Under "Source", select **"main"** from the branch dropdown
4. Click **"Save"**
5. Wait 1-2 minutes for deployment

## Step 3: Access Your App

1. GitHub will show you a URL like: `https://yourusername.github.io/repository-name`
2. Click that URL to open your app
3. Bookmark it!

## Step 4: Install on Your Phone

### iPhone/iPad:
1. Open the URL in Safari
2. Tap the **Share** button (square with arrow)
3. Scroll down and tap **"Add to Home Screen"**
4. Tap **"Add"**

### Android:
1. Open the URL in Chrome
2. Tap the **menu** (three dots)
3. Tap **"Add to Home Screen"**
4. Tap **"Add"**

## You're Done!

The app icon will appear on your home screen. Tap it to launch the app - it works just like a native app and will work offline!

## Troubleshooting

**App not loading?**
- Make sure all files were uploaded
- Check that GitHub Pages is enabled
- Wait a few minutes and refresh

**Can't add to home screen?**
- Make sure you're using Safari (iOS) or Chrome (Android)
- Some browsers don't support PWA installation
